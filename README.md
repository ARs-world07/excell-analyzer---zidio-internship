
#  Excel Data Analysis & Charting Tool

A simple web app to upload Excel files, view the data, and generate interactive charts.

##  Features

- Upload `.xlsx` files
- Preview data in a table
- Create bar, line, pie, scatter & doughnut charts
- Download charts as images
- Mock login/registration and upload history
- Clean UI with Tailwind CSS and Chart.js

##  Built With

- HTML, CSS (Tailwind), JavaScript
- SheetJS for Excel parsing
- Chart.js for data visualization

This is frontend only project
